# To-Do App With React/Redux


This project is based on [TodoMVC](http://todomvc.com)

Remade with React and Redux Toolkit.


### In this project, you can:
* Add
* Delete selected
* Toggle selected
* Toggle all
* Clear completed
* Update
* and Filter todos



## Tech Stack 
```
* React
* reduxjs/toolkit
```
